# ComfyUI 全能工具

AstrBot 插件 — 连接本地/云端 ComfyUI，实现文生图、图生图、图生视频、图片编辑等功能。自带 WebUI 可视化面板。

## 快速开始

### 前置条件

- 已部署 ComfyUI（本地或云端）
- 已安装 AstrBot
- 已将工作流 JSON 文件放入指定目录

### 安装插件

将插件文件夹放入 AstrBot 的 `data/plugins/` 目录，重启 AstrBot 即可。

### 基本用法

```
/画 一个白发Miku
```

在群内发送后，机器人自动生成图片并发送。

## 详细使用说明

### 🎨 `/画` — 文生图

```
/画 一个白发Miku
/画 16:9 一个白发Miku
```

- 第一个参数可选：比例（如 `9:16`、`1:1`、`4:3`）
- 剩余部分为提示词
- 支持直接发送纯文本触发（取决于工作流配置）

### 🖼️ `/图生图` — 图生图

```
/图生图 把背景改成红色
/图生图 0.7 把背景改成红色
```

- **引用图片**：回复/引用一张带图片的消息 + 发送命令
- **@用户头像**：@用户 + 发送命令（无引用图片时生效）
- 可选第一个参数为降噪值（`0.1` ~ `0.8`，默认由工作流决定）
- 降噪值越低，图片变化越小

### 🎬 `/图生视频` — 图生视频

```
/图生视频
```

- 需引用图片或 @用户获取头像
- 当前工作流需要是视频工作流（名称含 `视频`、`wan`、`ltx`、`animate`、`video`）
- 视频生成时间较长，请耐心等待

### 🎲 `/随机图` — 随机抽卡

```
/随机图
/随机图 5
```

- 只能在名称包含「抽卡」的工作流上使用
- 可选参数：抽卡张数（最多 10 张）
- 多张抽卡会依次执行并分别发送

### ✏️ `/编辑` — 图片编辑

```
/编辑 把这个猫换成狗
```

- 支持引用图片、@用户、上传图片（最多3张参考图）
- 图片会按顺序传入工作流的 LoadImage 节点
- 如果没有参考图会提示

### 📂 `/工作流` — 工作流管理

```
/工作流          → 显示所有工作流列表
/工作流 3        → 切换到编号3的工作流
/工作流 anime    → 搜索名称包含"anime"的工作流
/切换 anime      → 同上，快速切换
```

- 工作流按分类分组显示（画 / 图生图 / 图生视频 / 编辑 / 随机图）
- 选择后 10 秒内回复数字可切换
- 当前工作流标记 ✅

### 📐 `/比例` — 图片比例

```
/比例            → 显示所有可用比例
/比例 3          → 切换到编号3的比例
/比例 16:9       → 按名称切换
```

- 影响生成图片的宽高比
- 选择后 10 秒内回复数字可切换

### 🎚️ `/分辨率` — 质量等级

```
/分辨率           → 显示所有质量等级
/分辨率 2         → 切换到编号2的质量等级
/分辨率 1080p     → 按名称切换
```

- 等级：`480p`(SD) → `720p`(标清) → `1080p`(高清) → `2K`(超清) → `4K`(原画)
- 等级越高，图片越清晰但生成越慢

### 🚀 `/执行` — 执行工作流

```
/执行 提示词
```

- 执行当前工作流，不限制分类
- 适用于未设置分类的工作流或需要临时覆盖分类的场景
- 与 `/画` 行为类似，但不会自动切换工作流

### 📊 `/队列` — 查看队列

```
/队列
```

- 显示 ComfyUI 当前队列状态
- 运行中任务数 + 等待中任务数

### ⏹️ `/停止` — 停止生成

```
/停止
```

- 停止当前用户提交的正在生成的任务
- 不影响其他用户的任务

### ↩️ `/撤回` — 撤回消息

```
/撤回
```

- 撤回本插件在该会话中最近发送的一张图片/视频
- 需 OneBot API 支持（`send_msg` 返回 `message_id` 时可用）

### ❓ `/帮助` — 帮助

```
/帮助
```

- 显示当前工作流名称和所有可用命令

## WebUI 可视化界面

插件启动后自动在本地启动 WebUI：

```
WebUI: http://127.0.0.1:8898 (本机)
```

### 功能概览

| 区域 | 功能 |
|------|------|
| 顶部栏 | Logo、标题、主题切换、状态指示、队列、网络开关、透明度滑块、上传模式 |
| 工作流侧栏 | 查看/切换/搜索工作流 |
| 节点列表 | 查看工作流节点、编辑参数 |
| 提示词输入 | 输入正面/负面提示词 |
| 参数面板 | 设置质量、比例 |
| 进度条 | 显示生成进度和已运行时间 |
| 画廊 | 浏览已生成的图片，支持删除和撤回 |
| 设置面板 | 配置目录、ComfyUI 地址等 |

### 界面功能说明

#### 顶部栏

顶部栏分上下两行，由 0.5px 细线分隔：

- **上行**（品牌区）：Logo + "创作画室"标题 + 主题切换按钮（白天/夜间/背景/设置）
- **下行**（状态区）从左到右：
  - `●` 绿色脉冲圆点 — ComfyUI 连接状态
  - 连接状态文本
  - `队列: N` — 当前队列任务数
  - 当前工作流名称徽章
  - `📶` — 局域网 (LAN) 开关
  - `🌐` — IPv6 开关
  - `☁️` — 界面透明度滑块（30%~100%）
  - `🖥️` — 上传模式（本地 / 远程）

#### 界面透明度控制

顶部栏右侧的 ☁️ 滑块控制 WebUI 界面透明度：

- **范围**：30% ~ 100%（默认 100%）
- **实现**：CSS `color-mix()` 动态混合所有表面颜色
- **不受影响**：画廊悬浮按钮、背景图片、文字颜色
- 透明度保存在浏览器 localStorage，刷新后自动恢复

### 配置项

| 配置 | 说明 |
|------|------|
| 工作流目录 | 存放工作流 JSON 文件的目录 |
| 输出目录 | 生成图片的保存目录 |
| ComfyUI 输入目录 | 上传图片时写入的目录 |
| ComfyUI 地址 | ComfyUI 服务地址（默认 `127.0.0.1:8188`） |
| 上传模式 | 本地/远程（影响图片上传路径） |
| 局域网访问 | 开启后可通过局域网 IP 访问 WebUI |
| IPv6 访问 | 开启后可通过 IPv6 访问 WebUI |

### 局域网/云端访问

在 WebUI 设置面板中开启「局域网访问」后：
- 本地局域网：`http://你的局域网IP:8898`
- 云服务器：`http://你的服务器公网IP:8898`（需开放防火墙端口）

## 工作流配置

### 目录结构

```
工作流目录/
├── Anima动漫画图.json
├── FLUX漫转真人.json
├── 基础图生视频.json
├── 原json/                     # 非 API 格式的工作流备份
│   ├── Anima动漫画图.json
│   └── ...
```

### 命令关联

在 `workflow_config.json` 中配置命令与工作流的对应关系：

```json
{
  "__commands__": {
    "画": ["Anima动漫画图.json", "FLUX漫转真人.json"],
    "随机图": ["Anima随机抽卡.json"],
    "图生图": ["图生图工作流.json"],
    "图生视频": ["视频生成工作流.json"],
    "编辑": ["编辑工作流.json"]
  }
}
```

- 每个命令关联多个工作流，用户可以通过 `/工作流` 切换
- 工作流 JSON 的文件名（不含路径）作为唯一标识

### 分类显示

```json
{
  "__wf_categories__": {
    "Anima动漫画图.json": "画",
    "FLUX漫转真人.json": "画",
    "图生图工作流.json": "图生图"
  }
}
```

## 部署模式

### 本地模式

```
AstrBot ──HTTP──→ ComfyUI (127.0.0.1:8188)
  │
  └── WebUI (127.0.0.1:8898)
```

- 所有组件运行在同一台电脑
- 上传图片保存在 ComfyUI input 目录
- 适合个人使用

### 云端模式

```
AstrBot (云端) ──HTTP──→ ComfyUI (云端或另一台服务器)
  │
  └── WebUI (0.0.0.0:8898)
       ↓ 公网访问
  用户浏览器
```

- AstrBot 部署在云服务器
- ComfyUI 可同机部署或远程
- 输出图片通过 AstrBot 发送到消息平台

### 上传模式切换

通过 WebUI 设置面板的「上传模式」开关：
- **本地模式**：图片上传到 ComfyUI input 目录
- **远程模式**：图片上传到插件 upload 目录

## 常见问题

### Q: 生成失败/超时
- 检查 ComfyUI 是否正常运行
- 检查 ComfyUI 地址配置是否正确
- 检查 ComfyUI 是否加载了所需的模型

### Q: WebUI 无法访问
- 默认绑定 `127.0.0.1`，仅本机可访问
- 需要在 WebUI 设置中开启「局域网访问」
- 云服务器需要开放对应端口的安全组/防火墙

### Q: 画廊没有图片
- 检查输出目录配置是否正确
- 图片需要是插件生成并保存到输出目录的

### Q: 撤回不可用
- 撤回依赖 OneBot API 返回的 `message_id`
- 如果使用 `context.send_message()` 标准发送（非 OneBot 直连），则无法撤回

### Q: 进度条不显示阶段进度
- 部分 ComfyUI 版本不发送执行进度的 WebSocket 消息
- 进度条自动切换为计时器模式，不影响图片生成

### Q: 多个用户同时使用
- 插件支持多用户并发
- 每个用户的任务独立排队
- `/停止` 仅停止当前用户自己的任务

## LLM 工具调用

插件注册了以下 LLM 可调用工具（`@filter.llm_tool()`），供支持 Function Calling 的 LLM 自动调用：

| 工具名 | 功能 | 关键参数 |
|--------|------|---------|
| `comfyui_draw` | 文生图 | `prompt`, `ratio` |
| `comfyui_img2img` | 图生图 | `prompt`, `image_url`, `denoise` |
| `comfyui_video` | 图生视频 | `image_url`, `prompt` |
| `comfyui_edit` | 编辑图片 | `prompt`, `image_urls`（最多3张） |
| `comfyui_random` | 随机抽卡 | `count` |
| `comfyui_switch_workflow` | 切换工作流 | `keyword` |
| `comfyui_list_workflows` | 列出可用工作流 | 无 |
| `comfyui_get_current_workflow` | 获取当前工作流 | 无 |
| `comfyui_queue_status` | 查看队列状态 | 无 |
| `comfyui_stop` | 停止生成 | 无 |
| `comfyui_execute` | 执行工作流 | `prompt` |

### 图片处理说明

当用户通过消息发送图片时，插件会自动拦截 LLM 请求中的图片数据：

1. 将图片（base64 / HTTP URL）下载到临时目录
2. 以文本路径形式告知 LLM：`[用户发送了图片，已保存到: path]`
3. LLM 调用 `comfyui_img2img` / `comfyui_video` / `comfyui_edit` 等工具时传入该路径

> **适用场景**：纯文本模型（如 DeepSeek V4 Flash）不支持 `image_url` 消息类型时，此机制确保 LLM 仍能感知用户发图并调用对应工具。

临时文件在图片上传到 ComfyUI 后即时删除，另有定时清理兜底（每 1 小时清理超过 24 小时的文件）。

## 跨平台兼容

插件核心功能与消息平台无关，支持在以下平台运行：
- **QQ**（OneBot）：完整功能，含撤回
- **Telegram / Discord / 飞书** 等：除撤回外所有功能正常

> 注意：撤回需要消息平台 API 返回 `message_id`，非 OneBot 平台可能不支持。
